## Project Description
This is a Formula 1 Vacations website on different Grand Prix events.
The aplication features a Trip section where users can create and manage their own trips,
choose how many nights they stay and select hotel for a specific race circuit.

## Main Features
- View Formula 1 races and their details.
- Create, edit, and delete trips.
- Assign hotels and number of nights to each trip.

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server - Bootstrap

## Setup Instructions
1. Apply the Entity Framework Core migrations.
2. Run the project using the default configuration.
